# ♟️ Chess Master Pro 2026

Кросс-платформенные шахматы с ИИ, сохранением партий, рейтинговой системой и темами оформления. Работает прямо в браузере — ничего устанавливать не нужно!

## 🚀 Быстрый старт

Просто откройте `index.html` в любом современном браузере:

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/chess-master-pro.git
cd chess-master-pro
open index.html
```

## ✨ Возможности

### 🎮 Режимы игры
- **🤖 Игра с ИИ** — 3 уровня сложности (Лёгкий, Средний, Сложный)
- **👥 Два игрока** — играйте с другом на одном устройстве


### 💾 Сохранение партий
- Автосохранение завершённых игр
- Ручное сохранение в любой момент
- Загрузка сохранённых партий
- Экспорт/импорт всех игр в JSON
- Экспорт в формате **PGN**
- Копирование позиции в формате **FEN**

### 📊 Рейтинговая система (ELO)
- Начальный рейтинг: **1200**
- Динамический расчёт на основе сложности ИИ
- История изменений рейтинга
- Статистика: победы, поражения, ничьи, винрейт
- Рейтинговые звания от 🥉 Новичок до 🌟 Супергроссмейстер

### 🎨 Темы оформления (6 штук)
- **Тёмная** (по умолчанию)
- **Светлая**
- **Классика** (золотистые акценты)
- **Океан** (синие оттенки)
- **Лес** (зелёная палитра)
- **Закат** (оранжево-розовые тона)

### ⏱️ Контроль времени
- Без времени / 3 / 5 / 10 / 15 / 30 минут
- Визуальное предупреждение при < 60 сек

### 🔊 Звуковые эффекты
- Синтезированные звуки через Web Audio API
- Звуки хода, взятия, шаха, мата, рокировки, превращения

### ⌨️ Горячие клавиши
- `Esc` — выход в меню / закрыть настройки
- `Ctrl/Cmd + Z` — отменить ход

## 🛠️ Технологии

- **HTML5** — семантическая разметка
- **CSS3** — CSS Variables, Grid, Flexbox, анимации
- **Vanilla JavaScript** — ES6+, без фреймворков
- **Web Audio API** — синтез звуков
- **localStorage** — сохранение данных
- **Web Bluetooth API** — Bluetooth-соединение

## 🧠 ИИ (Искусственный интеллект)

- **Алгоритм Negamax** с альфа-бета отсечением
- **Таблицы позиционной оценки (PST)**
- Глубина поиска: Лёгкий=2, Средний=3, Сложный=4 полухода
